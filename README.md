## Getting Started

1. Fork the Project
2. Clone the repo

```sh
git clone https://github.com/abdulloooh/debtmanager.git
```

3. Install NPM packages

```sh
npm install
```

4. Set jwtPrivateToken from terminal e.g for Linux

```sh
export jwtPrivateToken=debtmanager
```

5. Run

```sh
npm run dev
```
